爱贷网爱心公益平台简介
爱贷网设立"爱心捐助"旨在倡导一方有难，八方支援的互助精神，使遭遇特殊困难的弱势群体感受到社会的关怀，感受到人与人之间的关爱。
1.捐助对象：遭遇重大疾病、意外伤害或特殊困难的个人和家庭。
2.项目来源：爱贷网的捐助对象来自于爱贷网办事处所在地农村或与微公益合作的项目，爱贷网对每一个受捐助对象的真实情况进行核实，符
合条件的才会上传至爱贷网。
3.捐助形式：目前只支持现金捐助，不接受实物。捐赠申请金额满额后，通过第三方支付平台将善款直接转入受捐人账户中，每一笔捐助明细
会员都可查询。
4.资金来源：爱贷网会员保证金（本金和利息收益）的自主捐助、社会各界爱心人士委托爱贷捐赠的资金、"微公益"基金项目提供的捐助。
5.捐助费用：对每一个独立的捐助收取所捐金额1%的费用，用于转账结算及档案管理。
